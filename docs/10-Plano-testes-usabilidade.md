# Plano de testes de usabilidade

<span style="color:red">Pré-requisitos: <a href="05-Projeto-interface.md"> Projeto de interface</a></span>, <a href="08-Plano-testes-software.md"> Plano de testes de software</a>

Este plano de testes de usabilidade tem como objetivo avaliar a qualidade da interface da aplicação SmartOuvidoria, verificando se ela é intuitiva, funcional e satisfatória para usuários com diferentes perfis e avaliar a facilidade de uso, eficiência e satisfação dos usuários ao interagirem com o chatbot SmartOuvidoria via WhatsApp. A análise será baseada em cenários reais de uso da aplicação, levando em consideração funcionalidades críticas definidas nas histórias de usuário e requisitos implementados.

## Seleção dos Participantes
# Perfil dos participantes: Usuários que representem os principais públicos da Ouvidoria Geral de Contagem:

-Cidadãos com baixo e médio grau de familiaridade tecnológica

-Funcionários da Ouvidoria

-Estudantes da área de TI (com conhecimento técnico)

Quantidade: 4 participantes

Critérios de seleção: Correspondência com os perfis de usuário identificados no projeto

Nenhum dado pessoal foi coletado, conforme as diretrizes da LGPD.

## Procedimentos de Teste
Formato dos testes: Presenciais e remotos

Método de coleta: Observação direta e medição

Ferramentas:

-Cronômetro digital (para tempo de execução)

-Formulário Google (para avaliação subjetiva)

-Gravação de tela (sem exibição de dados pessoais)

# Métricas Coletadas

Taxa de Sucesso: Se a tarefa foi concluída com sucesso (Sim / Não)

Tempo de Conclusão da Tarefa: Comparado ao tempo de execução por um especialista

Satisfação Subjetiva: Escala de 1 (péssimo) a 5 (ótimo)

## Cenários de teste de usabilidade

| Nº do Cenário | Descrição                                                                                                                             |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| 1             | Você é um cidadão que quer registrar uma denúncia relacionada a fiscalização de um terreno. Use o chatbot para fazer esse registro.   |
| 2             | Você já abriu um protocolo sobre capina, mas não obteve retorno. Use o chatbot para atualizar ou relatar a falta de resposta.         |
| 3             | Você quer saber quais documentos são necessários para registrar uma reclamação sobre IPTU. Use o chatbot para buscar essa informação. |
| 4             | Sua solicitação foi respondida, mas precisa ser redirecionada. Use o chatbot para pedir atendimento com um atendente humano.          |

## Resultados Esperados
Facilidade de uso para usuários não familiarizados com tecnologia

Identificação de gargalos na interação com o chatbot

Feedback direto sobre o conteúdo das respostas e navegação da conversa

Redução do tempo médio de registro de manifestações
